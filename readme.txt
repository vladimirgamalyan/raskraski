Чтобы выйти из программы нажмите клавишу Esc.
Для установки копируем всю папку куда-нибудь себе на диск.
Для запуска кликаем дважды по файлу raskraski.exe.
Для удобства можно создать ярлык на рабочем столе, для этого перетягиваем правой кнопкой
мыши файл raskraski.exe на рабочий стол и в момент отпускания кнопки появится меню, в
котором надо выбрать "создать ярлык".
